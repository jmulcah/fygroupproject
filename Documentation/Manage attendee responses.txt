Manage attendee responses
**requires an event to already have been created. Follow Create and edit events to do this**
**requires user to be logged in to admin account. Follow registering and logging in to do this**

1. Type 'bash run.sh' to run website
2. Open web browser to 'localhost:5000'
3. Click on events tab at top of page
**skip to step 7 if you want to add**
4. Click on guest list tab on desired event
5. Attendees can be removed from this page by clicking the x button on a given user
**stop here if not adding guests**
6. Click back to go to previous page
7. Click on the invite tab on desired event
8. Click the + button on each user you wish to add
