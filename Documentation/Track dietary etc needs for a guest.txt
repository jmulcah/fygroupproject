To track dietary etc needs for a guest:
**Must have created an admin first following the create admin.txt instructions. **
**Must have created an event first following the create and edit event.txt instructions **

1. Login in as admin user: admin@admin.com
		     pass: admin2018
2. Click on the event tab located at the top of the dashboard
3. Click on an 'view' under the guestlist column on an event
4. Click 'edit needs' under the edit special requirements column
5. Enter in details and click submit

*note* Special requirements of a user will be saved into the system for all future event
unless changed. As a diabetic will still be diabetic at another event.
